# UI Demo

This directory contains a standalone dashboard demo based on the provided `daed` screenshot.

## Files

- `index.html`: page structure
- `styles.css`: layout and visual style
- `script.js`: lightweight interactions and chart animation

## Run

Open `index.html` directly in a browser, or serve the directory locally:

```bash
cd ui
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
